Lists improve the readability and organization of information. Use them to break down complex instructions, features, or concepts into scannable pieces.

## Basic guidelines

- Use parallel writing structures for similar things.
- Use lists to present complex text in a way that's easy to scan.
- A list should have at least two items but no more than seven items.

## Unordered lists (bullets)

Use bulleted lists for items that share something in common but don't need to appear in a specific order.

<div className="not-prose grid grid-cols-1 md:grid-cols-2 gap-6 my-6">
  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-green-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-green-100 m-0">Correct</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p>The benefits of DevOps are:</p>
      <ul>
        <li>Faster deployment</li>
        <li>Improved collaboration</li>
        <li>Higher quality software</li>
      </ul>
    </div>
  </div>
  <div className="flex flex-col gap-3">
    <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
      <div className="px-5 py-3 bg-utility-red-20 border-b border-(--border)">
        <h3 className="text-base font-semibold text-utility-red-100 m-0">Incorrect</h3>
      </div>
      <div className="px-5 py-4 text-sm text-(--foreground)">
        <p>How to merge a pull request:</p>
        <ul>
          <li>Review the code changes.</li>
          <li>Run automated tests.</li>
          <li>Merge the pull request.</li>
        </ul>
      </div>
    </div>
    <p className="text-sm text-(--muted-foreground) m-0"><b>Why this is incorrect:</b> This process is sequential. Use a numbered list instead.</p>

## Ordered lists (numbered)

Use numbered lists for steps or items that must appear in a specific order.

    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <ol>
        <li>Turn on the computer.</li>
        <li>Log into your computer.</li>
        <li>Open the browser.</li>
      </ol>

## Introductory text

- Ensure the purpose of the list is clear. Introduce the list with a heading or a complete sentence.
- If you introduce a list with a heading, skip the explanatory text after the heading.
- Avoid using a colon or period in the heading.

<div className="not-prose grid grid-cols-1 md:grid-cols-2 gap-6 my-6">
  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-green-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-green-100 m-0">Correct</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p><strong>With heading:</strong></p>
      <h4>Steps to install XYZ</h4>
      <ol>
        <li>Step One.</li>
        <li>Step Two.</li>
      </ol>
      <p><strong>With sentence:</strong></p>
      <p>Follow these steps to install XYZ.</p>
      <ol>
        <li>Step One.</li>
        <li>Step Two.</li>
      </ol>
    </div>
  </div>
  <div className="flex flex-col gap-3">
    <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
      <div className="px-5 py-3 bg-utility-red-20 border-b border-(--border)">
        <h3 className="text-base font-semibold text-utility-red-100 m-0">Incorrect</h3>
      </div>
      <div className="px-5 py-4 text-sm text-(--foreground)">
        <p><strong>With heading + sentence:</strong></p>
        <h4 id="installation-steps">Installation Steps</h4>
        <p>The following steps guide you through the installation process.</p>
        <p><strong>With sentence but no clear purpose:</strong></p>
        <p>The software installation involves several steps.</p>
      </div>
    </div>
    <div className="flex flex-col gap-2">
      <p className="text-sm text-(--muted-foreground) m-0"><b>Why the first is incorrect:</b> A heading and an explanatory sentence together are redundant. Use one or the other.</p>
      <p className="text-sm text-(--muted-foreground) m-0"><b>Why the second is incorrect:</b> The sentence doesn't tell the reader what to expect. It doesn't introduce the list's purpose clearly.</p>
    </div>
  </div>
</div>

## Capitalization

- Begin each list item with a capital letter unless the item is a command, which is always lowercase.
- Rewrite items if needed so all items begin consistently — either all capitalized or all lowercase.

  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-green-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-green-100 m-0">Correct</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p><strong>Consistent capitalization</strong></p>
      <ul>
        <li>Download the application.</li>
        <li>Install the application.</li>
        <li>Launch the application.</li>
      </ul>
    </div>
  </div>
  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-red-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-red-100 m-0">Incorrect</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p><strong>Inconsistent capitalization</strong></p>
      <ul>
        <li>download the application.</li>
        <li>Install the application.</li>
        <li>Launch the Application.</li>
      </ul>
    </div>
  </div>
</div>

## Punctuation

- Apply punctuation consistently across all items in the list and all lists in the same section.
- Avoid semicolons, commas, or conjunctions such as *and* or *or* at the end of list items.
- Use a period only when list items are complete sentences. If all items have three or fewer words, omit the period.
- If the list is introduced by a sentence ending with a colon, end each item with a period. Each item should complete the introductory sentence.

<div className="not-prose grid grid-cols-1 md:grid-cols-2 gap-6 my-6">
  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-green-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-green-100 m-0">Correct</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p><strong>Complete sentences:</strong></p>
      <p>To secure your account:</p>
      <ul>
        <li>Enable two-factor authentication.</li>
        <li>Update your software regularly.</li>
      </ul>
      <p><strong>Short items without complete sentences:</strong></p>
      <p>Our priorities are:</p>
      <ul>
        <li>Quality</li>
        <li>Efficiency</li>
        <li>Innovation</li>
      </ul>
    </div>
  </div>
  <div className="rounded-lg border border-(--border) bg-(--card) overflow-hidden">
    <div className="px-5 py-3 bg-utility-red-20 border-b border-(--border)">
      <h3 className="text-base font-semibold text-utility-red-100 m-0">Incorrect</h3>
    </div>
    <div className="px-5 py-4 text-sm text-(--foreground)">
      <p><strong>Incomplete sentences with periods:</strong></p>
      <p>Improvements include:</p>
      <ul>
        <li>Faster loading time.</li>
        <li>Improved security.</li>
      </ul>
      <p><strong>Mixed punctuation:</strong></p>
      <p>Our priorities are:</p>
      <ul>
        <li>Quality,</li>

      </ul>
    </div>
  </div>
