:::note About this project
Modern documentation search indexes entire documents, burying users in loosely related results — and search-as-a-service tools like Algolia or Typesense don't work in firewall-restricted or offline environments. This Docusaurus 3.x plugin revives metadata-driven search, the same principle that made XML-based help systems effective two decades ago. Keywords defined in front matter drive results, keeping them focused and intentional — no external APIs, no network dependency, no noise. Built, published to npm, and documented as a solo open-source project.
:::
